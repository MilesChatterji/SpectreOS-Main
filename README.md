# SpectreOS

An opinionated NixOS derivative targeting people who want the power of NixOS without the from-scratch learning curve. Focused on aesthetics, privacy, and creativity.

Built on NixOS 25.11 with kernel 7.0, the [Niri](https://github.com/YaLTeR/niri) Wayland compositor, and [Noctalia Shell](https://github.com/bedsteler20/noctalia).

> **Status: pre-installer.** The configuration is stable and daily-driver quality on the reference hardware (ASUS ProArt PX13). The installer — the milestone that earns the derivative OS label — is the current development priority. Until it ships, this is a sophisticated personal configuration with a vision and a roadmap.

---

## Design goals

- A custom installer that produces a complete SpectreOS experience from scratch
- Hardware-agnostic base with clean per-host layering
- Original UI work via Noctalia contributions or fork
- A GUI updater that abstracts `nixos-rebuild switch` behind something approachable
- Coherent branding and a target audience — not a general-purpose NixOS config

## Repo structure

```
configuration.nix          # Hardware- and user-agnostic base
modules/
  desktop/
    niri.nix               # Niri compositor + Noctalia Shell + auto-brightness
hosts/
  px13/                    # ASUS ProArt PX13 (reference hardware)
    default.nix            # Entry point — imports base + all px13 modules
    user.nix               # User account (PX13-specific groups for DialPad)
    gpu-offload.nix        # AMD iGPU phase 1 / NVIDIA phase 2
    asus-dialpad.nix       # ASUS DialPad driver
    hardware-configuration.nix
  vm/                      # QEMU/KVM install target for testing
    default.nix            # Entry point — imports base + SSH + qemu-guest
    user.nix               # User account
    hardware-configuration.nix   # generated by nixos-generate-config, not committed
assets/                    # Plymouth boot splash images
notes/                     # Design notes, hardware research, migration logs
```

## VM install (testing)

1. Create a QEMU/KVM VM and boot the [NixOS installer ISO](https://nixos.org/download/)
2. Partition and mount disks, then run:
   ```
   nixos-generate-config --root /mnt
   ```
3. Clone this repo into `/mnt/etc/nixos/`
4. Replace `/mnt/etc/nixos/configuration.nix` with a symlink or reference to `hosts/vm/default.nix`
5. Edit `hosts/vm/user.nix` with your username and details
6. Run `nixos-install` and reboot

SSH is enabled in the VM profile (`services.openssh`). Default password is `spectreos` — change it after first boot.

## Reference hardware

The PX13 host (`hosts/px13/`) is the primary development and daily-driver machine:

- ASUS ProArt PX13 (2024)
- AMD Ryzen AI 9 HX 370 / Radeon 890M iGPU
- NVIDIA GeForce RTX 4070 (PRIME offload — phase 2, pending VM validation)
- ASUS DialPad (custom Python driver via `asus-dialpad.nix`)
- Ambient light sensor (auto-brightness via `modules/desktop/niri.nix`)

---

## Roadmap

1. **Barebones installer** — the current priority. Boots, partitions, installs, sets up a user.
2. **iGPU-only v1** — ships without NVIDIA. Covers most modern laptops, testable in VMs.
3. **NVIDIA PRIME offload** — added after a clean iGPU install is validated.
4. **Noctalia contributions** — upstream PR first; fork if needed.
5. **GUI updater** — developed in parallel with the installer.
